# 書類スキャンシステム 運用マニュアル

---

## 1. 新しい顧問先を追加する

### 手順

**① Apps Scriptを開く**

https://script.google.com を開き、「書類スキャン」プロジェクトを開く

**② 顧問先名を入力する**

コードの中から以下の行を探す（239行目あたり）：

```javascript
const clientName = 'ここに顧問先名を入力';
```

`ここに顧問先名を入力` を顧問先名に書き換える：

```javascript
const clientName = '田中商事';
```

**③ 実行する**

- 上部の関数選択で `registerClient` を選ぶ
- 「実行」ボタンを押す

**④ URLを確認する**

- 下の「実行ログ」に配布用URLが表示される
- このURLをコピーして顧問先に渡す

```
========================================
登録完了: 田中商事

配布用URL:
https://yukihiro-jpg.github.io/test-project/frontend/?client=xxx&name=田中商事
========================================
```

**⑤ 次の顧問先を登録する場合**

②の名前を書き換えて、もう一度③を実行するだけ

## 3. 毎日の確認作業

### 毎朝やること

**① Gmailを確認する**

- 毎朝6時に「【書類スキャン】○件の新規アップロード」というメールが届く
- 前日にアップロードがなければメールは届かない

**② メール内のリンクからファイルを確認する**

- メール内の「開く」リンクで個別のPDFを確認
- 「Google Driveフォルダを開く」で全体を確認

**③ PDFをAIで解析する**

- Google Driveから必要なPDFをダウンロード
- AIツールで仕訳データに変換

## 6. 顧問先への案内テンプレート

以下をコピーして顧問先にメールやLINEで送る：

```
【書類スキャンのご案内】

下記URLをスマートフォンで開いてください。
（ブックマークしておくと便利です）

URL: ここにURLを貼り付け

【使い方】
1. URLを開く
2. 書類の種類を選ぶ（通帳の場合は銀行名も入力）
3.「撮影する」をタップして書類を撮影
4. 複数ページある場合は続けて撮影
5. 全て撮り終わったら「まとめて送信する」をタップ

※ アプリのインストールは不要です
※ 多少斜めでも自動補正されますが、なるべく明るい場所で撮影してください
※ Googleアカウントは不要です
```

## 8. トラブル対応

### 顧問先から「撮影できない」と言われた

- スマホのブラウザでカメラの許可を求められたら「許可」を押してもらう
- URLが正しいか「顧問先URL一覧」シートで確認する

### ファイルがDriveに届かない

- Apps Scriptエディタの「実行数」（左メニュー時計アイコン）で `doPost` のエラーを確認
- 再デプロイが必要な場合：「デプロイ」→「デプロイを管理」→ ✏️ →「新バージョン」→「デプロイ」

### メール通知が届かない

- Apps Scriptエディタで `testSendEmail` を選んで「実行」→ 手動でメール送信テスト
- 迷惑メールフォルダを確認

### Apps Scriptのコードを更新した場合

保存（Ctrl+S）だけでは反映されない。必ず再デプロイが必要：

```
「デプロイ」→「デプロイを管理」→ ✏️ →「新バージョン」→「デプロイ」
```
