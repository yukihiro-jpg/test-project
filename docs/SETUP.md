# 書類スキャンシステム - セットアップガイド

## 概要

顧問先がスマホで書類を撮影 → 自動台形補正 + OCR → Google Driveに保存 → 毎朝メール通知

## フォルダ構成

```
顧問先_書類スキャン/               ← ルートフォルダ
├── 田中商事/                     ← 顧問先フォルダ（自動作成）
│   ├── 20260325_1430_通帳_三菱UFJ銀行_5p.pdf
│   └── 20260326_0900_レシート・領収書_3p.pdf
├── 鈴木工業/
│   └── 20260325_1100_売上請求書_2p.pdf
└── _書類スキャン管理              ← 管理用スプレッドシート（自動作成）
    ├── アップロード履歴シート
    └── 顧問先URL一覧シート        ← URLはここで確認
```

## セットアップ手順

### Step 1〜3: 初期設定（前回と同じ）

1. Google Driveにフォルダ作成
2. Google Apps Scriptにコードを貼り付け・デプロイ
3. GitHub Pagesでフロントエンドを公開

### Step 4: 顧問先の登録（新方式）

**管理スプレッドシートから登録するだけでOK:**

1. Apps Scriptエディタを開く
2. 関数選択で `registerClient` を選ぶ
3. 「実行」をクリック
4. ダイアログに顧問先名を入力（例: 田中商事）
5. 自動で以下が行われます：
   - Google Driveに顧問先フォルダを作成
   - 配布用URLを生成
   - 「顧問先URL一覧」シートに記録

### 顧問先URLの再確認

顧問先からURLを聞かれたら：
1. Google Driveの `_書類スキャン管理` スプレッドシートを開く
2. 「顧問先URL一覧」シートを確認

## 顧問先の操作フロー

1. URLを開く → 画面上部に自分の会社名が表示される
2. 書類の種類を選ぶ（通帳の場合は銀行名も入力）
3. 「撮影する」をタップ → 1ページずつ撮影
4. 全ページ撮り終わったら「まとめて送信する」をタップ
5. 複数ページが1つのPDFとして送信される
