# 営業成績ダッシュボード

税理士が顧問先の中小企業社長向けに営業成績の報告資料を作成するためのツールです。
会計ソフトからエクスポートした元帳データ（CSV）を読み込み、インタラクティブなダッシュボードで分析結果を表示します。

## 機能

- **KPIサマリー**: 売上高・売上総利益・営業利益・当期純利益を前期比付きで表示
- **当期月次推移**: 月別の売上・利益推移チャート、累計推移、予算対比
- **3期比較**: 当期・前期・前々期のP/L横並び比較テーブルとグラフ
- **前年対比**: 月次ベースの当期vs前期比較、増減率分析
- **利益率・費用構成**: 利益率推移、P/Lウォーターフォール、販管費内訳
- **キャッシュフロー**: 営業CF/投資CF/財務CFの推移と現金残高
- **借入金返済**: 借入金残高推移、返済スケジュール、完済シミュレーション（繰上返済対応）
- **損益分岐点**: 損益分岐点売上高、安全余裕率、固変分解
- **エクスポート**: PDF報告書 / Excelダウンロード

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

## 元帳CSVフォーマット

会計ソフトからエクスポートしたCSVファイルを使用します。

### 必須列

| 列名 | 説明 |
|------|------|
| 日付 | 取引日（YYYY/MM/DD形式） |
| 勘定科目 | 勘定科目名 |
| 借方金額 | 借方金額 |
| 貸方金額 | 貸方金額 |

### 任意列

| 列名 | 説明 |
|------|------|
| 補助科目 | 補助科目名 |
| 摘要 | 取引の説明 |
| 部門 | 部門名（部門別分析に使用） |

### 対応エンコーディング

UTF-8、Shift-JIS、CP932 を自動判別します。

### 予算CSV（任意）

| 列名 | 説明 |
|------|------|
| 年月 | YYYY/MM形式 |
| 勘定科目 | 勘定科目名 |
| 予算額 | 月次予算額 |

## 対応勘定科目

売上高、仕入高、各種販管費、営業外収益/費用、特別利益/損失、法人税等、
現金・預金、売掛金、買掛金、長期借入金、短期借入金など40科目以上に対応。
未分類の科目がある場合は画面上に警告表示されます。
