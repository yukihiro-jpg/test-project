# MJS電子申告PDF分割ツール (mjs-pdf-split)

MJSの電子申告システムからダウンロードした結合PDFを、書類種類ごとに自動分割し、わかりやすいファイル名にリネームするツールです。

## 特徴

- 結合PDFを書類種類ごとに自動分割
- ファイル名を `{書類名}_{決算期}_{会社名}.pdf` に自動リネーム
- 完全ローカル処理（クラウド通信なし）
- Pythonなしで動作するスタンドアロン実行ファイル対応

| 消費税申告書 | `消費税申告書_令和6年3月決算_サンプル株式会社.pdf` |

| 事業概況説明書 | `事業概況説明書_令和6年3月決算_サンプル株式会社.pdf` |
| 県税申告書 | `県税申告書_令和6年3月決算_サンプル株式会社.pdf` |
| 市税申告書 | `市税申告書_令和6年3月決算_サンプル株式会社.pdf` |

## 使い方

### スタンドアロン実行ファイル（推奨）

事務所内のどのPCでもPythonインストール不要で使用できます。

```
mjs-pdf-split.exe 電子申告ダウンロード.pdf
```

### 複数ファイルの一括処理

```
mjs-pdf-split.exe file1.pdf file2.pdf file3.pdf
```

### オプション

```
mjs-pdf-split [OPTIONS] INPUT_PDF...

オプション:
  --output-dir DIR   出力先ディレクトリを指定（デフォルト: 入力ファイルと同じ場所にサブフォルダ作成）
  --dry-run          実際にファイルを作成せず、分割結果のプレビューのみ表示
  --verbose          詳細なログを表示
  --version          バージョン表示
  --help             ヘルプ表示
```

## 開発者向け

### インストール（Python環境）

```bash
pip install .
```

### テスト実行

```bash
pip install ".[dev]"
pytest
```

### スタンドアロン実行ファイルのビルド

```bash
pip install ".[dev]"
pyinstaller build.spec
```

`dist/mjs-pdf-split.exe` が生成されます。このファイルを事務所内のPCに配布してください。

## 注意事項

- テキストベースのPDFのみ対応（スキャン画像PDFは非対応）
- MJS電子申告システムからダウンロードしたPDFを対象としています
- すべての処理はローカルで完結し、ネットワーク通信は一切行いません
