# 会計大将インポートCSV生成ツール

## 概要
このツールは顧問先の資料（通帳PDF、売上請求書PDF、賃金台帳PDF等）を読み取り、
会計大将にインポートするための26列CSVファイルを自動生成するPythonツールです。

## Claude Codeへの指示

### PDF読み取りのカスタマイズ
事務所PCで実際のPDFをClaude Codeに見せる際、以下の手順で対応してください：

1. `modules/pdf_reader.py` の該当するparse関数を開く
2. 実際のPDFの構造（テーブルレイアウト、テキスト配置）を分析
3. 正規表現パターンや列位置を実際のPDFに合わせて調整
4. 顧問先ごとに異なるフォーマットの場合は、ファイル名やフォルダで判別するロジックを追加

### 仕訳精度向上のサイクル
1. `python main.py 顧問先名` で仕訳CSV生成
2. 生成された `review_*.csv` で要確認項目を人間が確認
3. 確認結果を基にルールブック（rulebook.txt）を手動または自動で更新
4. 次回以降の精度が向上

### フォルダ構成
```
kaikei_tool/
├── main.py              # メインスクリプト（CLI）
├── config.py            # 共通設定
├── requirements.txt     # Python依存ライブラリ
├── setup.bat            # Windowsセットアップ
├── modules/
│   ├── pdf_reader.py    # PDF読み取り（★カスタマイズ対象）
│   ├── master.py        # マスタ管理
│   ├── rulebook.py      # ルールブック管理
│   ├── csv_exporter.py  # CSV出力
│   └── journal_engine.py # 仕訳判定エンジン
└── clients/
    └── 顧問先名/
        ├── rulebook.txt     # ルールブック辞書
        ├── 過去仕訳/         # 過去の仕訳エクスポートCSV
        ├── 当月資料/         # 通帳PDF、請求書PDF等
        ├── マスタ/           # 科目リスト、補助科目等
        └── output/          # 生成されたCSV
```

### 会計大将CSV 26列フォーマット
日付, 借方科目コード, 借方科目名, 借方補助コード, 借方補助名,
借方税売仕区分, 借方業種, 借方税込税抜, 借方金額, 借方消費税額,
借方消費税コード, 借方消費税率コード, 借方事業者区分,
貸方科目コード, 貸方科目名, 貸方補助コード, 貸方補助名,
貸方税売仕区分, 貸方業種, 貸方税込税抜, 貸方金額, 貸方消費税額,
貸方消費税コード, 貸方消費税率コード, 貸方事業者区分, 摘要
